# Voting

![Voting Program](/images/Voting 23.png?raw=true "Voting Program")


This project contains a Javascript program written to explore various voting methods. The program includes:

- Simple Plurality
- Range Voting
- Instant Runoff
- Condorcet

The Voting program can be run here:
